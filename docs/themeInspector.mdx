### Theming

The inspector and the controls can be themed by passing an object as shown below to the `theme` prop of the inspector component.


```javascript
const CustomTheme = {
  primaryColor: "string",
  backgroundColor: "string",
  collapseHandle: {
    backgroundColor: "string",
  },
  knob: {
    label: {
      color: "string",
      dividerColor: "string",
    },
    slider: {
      railColor: "string",
      handleColor: "string",
      activeTrackColor: "string",
    },
  },
  log: {
    objectViewerTheme: "string",
    label: {
      color: "string",
      dividerColor: "string",
    },
  },
}
```
